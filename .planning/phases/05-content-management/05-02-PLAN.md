|-----------|----------|-----------|-------------|-----------------|
| T-05-03 | Elevation of Privilege | createDepartmentAction called by non-admin | mitigate | requireCmsRole() checks profile.role against CMS_ROLES before any DB write; throws Forbidden if not admin/super_admin |
| T-05-04 | Tampering | XSS via description/name stored in DB | accept | Plain text fields only; React escapes output by default; no dangerouslySetInnerHTML used |
| T-05-05 | Spoofing | Unauthenticated Server Action call | mitigate | requireCmsRole() calls getUser() (re-validates with Auth server) and redirects to /login if no session |
| T-05-SC | Tampering | shadcn add installs | accept | shadcn official registry; no unknown third-party packages |
</threat_model>
