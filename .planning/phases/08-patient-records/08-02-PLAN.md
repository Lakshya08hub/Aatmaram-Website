    - updatePatientAction(id, formData): same roles; updates all fields except clinical_notes

  <action>
    Create app/(portal)/actions/patients.ts with 'use server' directive.

    Define two Zod schemas:
    - receptionistPatientSchema: z.object({ patient_name: z.string().min(2), age: z.coerce.number().int().min(0).max(150), phone: z.string().min(7), reason: z.string().min(3), assigned_doctor_id: z.string().uuid().nullable().optional(), visit_date: z.string() })
    - doctorNotesSchema: z.object({ clinical_notes: z.string() })

    Helper requirePatientWriteRole(): reads auth session via createClient().auth.getUser(), looks up profiles.role by user_id, throws Error('Forbidden') if role is 'doctor'. Returns { user, role, supabase }.

    Helper requireDoctorRole(): same lookup; throws Error('Forbidden') if role is NOT 'doctor'. Returns { user, role }.

    Export async createPatientAction(prevState: unknown, formData: FormData): Promise<{ error?: string }>
    - Calls requirePatientWriteRole().
    - Parse formData fields using receptionistPatientSchema.safeParse(Object.fromEntries(formData)).
    - If invalid, return { error: validation.error.issues[0].message }.
    - createAdminClient().from('patient_records').insert(validated data — omit clinical_notes).
    - Return {} on success.

    Export async updatePatientAction(id: string, prevState: unknown, formData: FormData): Promise<{ error?: string }>
    - Same gate as createPatientAction.
    - Parse with receptionistPatientSchema.
    - createAdminClient().from('patient_records').update(validated data).eq('id', id).
    - Return {} on success.

    Export async updateClinicalNotesAction(id: string, prevState: unknown, formData: FormData): Promise<{ error?: string }>
    - Calls requireDoctorRole() to get { user }.
    - Resolve doctorId: adminClient.from('doctors').select('id').eq('staff_user_id', user.id).single().
    - Fetch record: adminClient.from('patient_records').select('assigned_doctor_id').eq('id', id).single(). Verify record.assigned_doctor_id === doctorId — return { error: 'Forbidden' } if not.
    - Parse formData with doctorNotesSchema.
    - adminClient.from('patient_records').update({ clinical_notes: validated.clinical_notes }).eq('id', id).
    - Return {} on success.
  </action>
