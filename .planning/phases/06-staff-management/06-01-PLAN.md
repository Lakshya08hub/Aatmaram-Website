    Create app/(portal)/actions/staff.ts ('use server') exporting:

    1. Helper `requireAdminRole()` — same pattern as requireCmsRole() in content.ts:
       - createClient(), getUser(), fetch profile role
       - Redirect to /login if no session
       - Throw Error('Forbidden') if role not in ['super_admin', 'admin']
       - Return supabase client

    2. `createStaffAction(input: { email, temp_password, role, full_name, phone?, salary?, join_date? }): Promise<{ error?: string }>`:
       - Call requireAdminRole() first
       - Use createAdminClient().auth.admin.createUser({ email, password: temp_password, email_confirm: true })
       - If Auth error → return { error: authError.message }
       - Insert into profiles via regular client: { user_id, role, is_active: false, full_name, phone, salary, join_date }
       - If profile insert error → call adminClient.auth.admin.deleteUser(authData.user.id) as rollback, return { error }
       - Return {}

    3. `updateStaffAction(id: string, input: { role?, full_name?, phone?, salary?, join_date?, is_active? }): Promise<{ error?: string }>`:
       - Call requireAdminRole()
       - Update profiles set {...input} where id = id
       - Return { error? }

    4. `toggleActiveAction(id: string, is_active: boolean): Promise<{ error?: string }>`:
       - Call requireAdminRole()
       - Update profiles set { is_active } where id = id
       - Return { error? }

    5. `deleteStaffAction(id: string, user_id: string): Promise<{ error?: string }>`:
       - Call requireAdminRole()
       - Delete from profiles where id = id (profiles row first)
       - Call createAdminClient().auth.admin.deleteUser(user_id) to remove Auth account
       - Return { error? }
  </action>
