    Export an async POST function accepting NextRequest:

    Step 1 — Parse and validate request body:
    Parse JSON body as { messages }. Validate that messages is an Array. Validate that each element has role (string, must be 'user' or 'assistant') and content (string). If validation fails, return new Response(JSON.stringify({ error: 'Invalid request' }), { status: 400, headers: { 'Content-Type': 'application/json' } }).

    Step 2 — Limit history depth to prevent cost amplification:
    Slice messages to the last 20 items before processing: const safeMessages = messages.slice(-20).

    Step 3 — Map to Gemini content format:
    Map safeMessages to Gemini's content format: each item becomes { role: m.role === 'assistant' ? 'model' : 'user', parts: [{ text: m.content }] }. The role mapping (assistant → model) is CRITICAL — Gemini rejects 'assistant' as a role name (see Pitfall 2 in RESEARCH.md).

    Step 4 — Call Gemini with streaming:
    Call ai.models.generateContentStream({ model: 'gemini-2.5-flash', contents: mappedContents, config: { systemInstruction: HOSPITAL_KNOWLEDGE } }).

    Step 5 — Pipe to ReadableStream:
    Create a new ReadableStream with an async start(controller) function. Inside, use a TextEncoder. Iterate the stream with `for await (const chunk of stream)`. On each chunk, access chunk.text — if it is truthy, encode and enqueue it. In a finally block, call controller.close(). Wrap the for-await in try/catch; on error, call controller.error(err).

    Step 6 — Return Response:
    Return new Response(readable, { headers: { 'Content-Type': 'text/plain; charset=utf-8', 'X-Content-Type-Options': 'nosniff' } }).

    The entire POST handler must be wrapped in a try/catch. On any unhandled error, return new Response(JSON.stringify({ error: 'Internal server error' }), { status: 500, headers: { 'Content-Type': 'application/json' } }).

    TypeScript note: The messages parameter from the request body should be typed as Array<{ role: string; content: string }> to satisfy the validator before narrowing.
