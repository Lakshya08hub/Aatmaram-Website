## Widget Anatomy

### Closed State: Floating Bubble
- Position: `fixed bottom-8 right-12` (32px bottom, 48px right)
- Size: 56px × 56px circular button
- Icon: `MessageCircle` from lucide-react, 24px, white
- Background: `#1E40AF` (primary blue)
- Shadow: `shadow-lg` (Tailwind)
- Hover: brightness-110, scale-105 transition (150ms ease)
- z-index: `z-50`

### Open State: Chat Panel
- Position: `fixed bottom-24 right-12` — floats above bubble with 16px gap
- Size: 360px wide × 500px tall
- Shadow: `shadow-xl`
- Border radius: `rounded-xl` (matches shadcn Card default via `--radius`)
- z-index: `z-50`
- Animation: slide-up + fade-in on open (tw-animate-css: `animate-in slide-in-from-bottom-4 fade-in duration-200`)

### Panel Header
- Background: `#1E3A5F` (`--color-nav`)
- Height: 56px
- Left: Hospital name "Atmaram Child Care" in white, 15px, weight 600
- Right: X close button — `Button variant="ghost"` with `X` icon (16px) in white/70 opacity, hover white
- Padding: `px-4`

### Message List (scrollable)
- Background: `#F8FAFC`
- Padding: `p-4`
- Gap between message groups: `gap-3` (12px — between consecutive bubbles from same sender), `gap-4` (16px — between sender switches)
- Overflow: `overflow-y-auto`
- Scroll behavior: auto-scroll to bottom on new message

### Message Bubbles
- **User bubble:** right-aligned, `bg-primary` (#1E40AF), white text, `rounded-2xl rounded-br-sm`, `px-4 py-2`, max-width 80%
- **Bot bubble:** left-aligned, `bg-slate-100`, `--color-text` (#0F172A) text, `rounded-2xl rounded-bl-sm`, `px-4 py-2`, max-width 80%
- Body text: 14px / weight 400 / line-height 1.5

### Typing Indicator (while waiting for first token)
- Shown in a bot bubble position (left-aligned)
- Three dots: `bg-primary` (#1E40AF), 8px circles, animated with staggered `animate-bounce` (Tailwind)
- Dot 1: delay 0ms, Dot 2: delay 150ms, Dot 3: delay 300ms
- Hidden as soon as first streaming token arrives

### Message Composer (bottom bar)
- Background: `#FFFFFF`
- Border top: `border-t border-border`
- Padding: `p-3`
- Layout: `flex gap-2`
- Input: `Input` component, placeholder "Type a message...", `flex-1`
- Send button: `Button size="icon"`, `bg-primary`, `SendHorizonal` icon (16px), white
- Send button disabled state: `opacity-50 cursor-not-allowed` when input is empty or streaming in progress

## Interaction States

| State | Trigger | Visual |
|-------|---------|--------|
| Bubble idle | Default | Blue circle, MessageCircle icon |
| Bubble hover | Mouse over | Scale 1.05, brightness +10%, 150ms ease |
| Panel opening | Bubble click | slide-in-from-bottom-4 + fade-in, 200ms |
| Panel closing | X click or bubble click | fade-out, 150ms |
| Greeting shown | Panel first opens | Pre-set bot bubble appears instantly, no API call |
| User sends message | Enter key or Send button click | User bubble appended; input cleared; typing indicator shown |
| Streaming in progress | First token received | Typing indicator hidden; bot bubble appears and grows token by token |
| Streaming complete | Stream ends | Send button re-enabled |
| Error state | API failure | Bot bubble with error copy (see Copywriting) |
| Send button disabled | Empty input OR streaming active | `opacity-50`, pointer-events-none |

## Copywriting Contract

| Element | Copy |
|---------|------|
| Greeting message (bot, on open) | "Hello! I can help with information about Atmaram Child Care and Critical Care. Ask me about our departments, doctors, OPD timings, or how to book an appointment." |
| Hindi greeting (auto, if user writes in Hindi first) | Gemini handles language match automatically — greeting is always English as it precedes user input |
| Input placeholder | "Type a message..." |
| Send button aria-label | "Send message" |
| Close button aria-label | "Close chat" |
| Bubble button aria-label | "Open Atmaram chat assistant" |
| Error state (API failure) | "Something went wrong. Please try again, or call us at [hospital phone number]." |
| Out-of-scope fallback (handled by Gemini system prompt, not UI) | System prompt instructs Gemini to reply with the hospital phone number — not a UI copy element |
| Empty message (send guard) | No toast needed — send button is disabled when input is empty |
| Streaming cursor | No explicit cursor text — streaming tokens append directly |

No destructive actions in this phase. No confirmation dialogs required.

## Accessibility

- Bubble button: `aria-label="Open Atmaram chat assistant"`, `aria-expanded={isOpen}`
- Panel: `role="dialog"`, `aria-label="Atmaram chat assistant"`, `aria-modal="false"` (widget, not blocking)
- Message list: `aria-live="polite"` — screen readers announce new bot messages
- Close button: `aria-label="Close chat"`
- Send button: `aria-label="Send message"`, `disabled` attribute when inactive
- Focus management: when panel opens, focus moves to Input field; when panel closes, focus returns to bubble button
- Keyboard: Enter key in Input submits message (unless Shift+Enter for newline — not supported in v1, single-line input only)
